# 📚 Library Management System

A web-based Library Management System developed using Flask, HTML, CSS, and SQLite.
This application helps users manage library books efficiently with features like adding, editing, deleting, and viewing books through a clean and responsive interface.

---

## 🚀 Features

* Add new books
* Edit existing books
* Delete books
* View all books
* Responsive user interface
* SQLite database integration
* Simple and user-friendly dashboard

---

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3

### Backend

* Python
* Flask

### Database

* SQLite
